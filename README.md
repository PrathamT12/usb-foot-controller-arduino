# USB Foot Controller for Amp Simulator Software

## Overview
This project implements a **USB MIDI/foot controller** using an **Arduino Micro (ATmega32u4)**.  
It allows real-time interaction with PC-based guitar amp simulator software, providing a **reliable, event-driven interface** for live performances.

---

## Features
- Event-driven input handling for 5 footswitches  
- Software debouncing and state management to avoid false triggers  
- USB MIDI communication with host applications  
- Tested for reliable, real-time performance  

---

## Technical Details
- **Microcontroller:** Arduino Micro (ATmega32u4)  
- **Programming Language:** C / C++ (Arduino IDE)  
- **Libraries:** MIDIUSB  
- **Design Focus:** Event-driven software, state-based logic, USB communication
